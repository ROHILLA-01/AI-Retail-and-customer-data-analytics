# AI-Retail-and-customer-data-analytics
Built an AI-powered retail and customer analytics platform using Python, MySQL, Machine Learning, and Power BI to analyze customer behavior, predict sales trends, perform customer segmentation, and generate interactive business intelligence dashboards. Python + MySQL + Power BI + Machine Learning
The project simulates a real-world retail analytics pipeline used by e-commerce and retail companies for business intelligence and customer analytics.

projective outcomes:
The main objectives of this project are:
Analyze retail transaction data
Perform customer behavior analysis
Identify top-performing products and store locations
Predict future sales trends
Segment customers using machine learning
Generate interactive Power BI dashboards
Create business insights for decision-making
